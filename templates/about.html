About

What is this application?

 A fully functional web application meant to simulate database functionality for the employees of Dunder Mifflin, a fictional company in the show The Office (U.S. Version)

How does it work?

 Using a Flask framework, the application's core logic is written in Python, which sets up a connection to the database running on ix-dev.cs.uoregon.edu. From there the user can interact with the database, viewing information about the company and its sales, or entering an order. There are a variety of other cool features about this applications, including AJAX calls, mobile-responsive frameworks, and so much more, which can be examined by looking at the code for the application. How do I use it?

 While the user guide describes the application in detail, some of the cool features of it are:

 Human Resource Tools
 Comprehensive tools for the human resources team at Dunder Mifflin.
  • Visit the Employees page to view detailed employee information, filtering by department if desired.
  • Visit the Salaries page to view company-wide salary information, including individual and departmental.

 Sales Tools
 Tools that enable the sales team at Dunder Mifflin
  • Help clients place an order through the Place Order page.
  • View the order history for a client in a given date range on the Order History page to make the customized sale solutions that you need.
  • View the popularity of all products in terms of total quantity and revenue through the Products page to view the performance of various stock items.

 Administrative Tools
 For helping upper management make informed decisions
  • View the performance of various clients through the Sales Regions page, which breaks down revenue for all clients within a sales region.
  • View the order history for a client in a given date range on the Order History page to make the customized sale solutions that you need.
  • View information on individual sales employees on the Sales Employees page, including total units sold and total revenue produced, ensuring that they are meeting performance standards.
